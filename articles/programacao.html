O que são microcontroladores?

Tanenbaum classificou os computadores em diversos tipos, entre eles estão os microcontoladores, mas afinal, o que são microcontroladores? Microcontroladores são computadore com processamento limitado que servem para sistemas especificos que não requerem muito processamento. Os microcontorladores são usados principalmente em sistemas simples de automação, a principal vantagem de usar esse tipo de computador é o baixo custo, os microcomputadores costumam ser mais baratos que os computadores convencionais de uso pessoal.
